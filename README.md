# Анализ антропометрических данных легкоатлетов, участвующих в олимпиаде с 1896 года по 2016 год.

## Цель проекта:
Исследовать взаимосвязь между антропометрическими данными (рост, вес, ИМТ) спортсменов и их успехами в различных дисциплинах легкой атлетики на Олимпийских играх с 1896 по 2016 год.

## Основные вопросы исследования:
1. Как различаются рост и вес спортсменов в разных дисциплинах?
2. Есть ли эволюция физических параметров олимпийцев за 120 лет?
3. Наблюдаются ли четкие кластеры антропометрических данных для разных групп дисциплин?


## Что было сделано:

### 1. Сбор и первичный анализ данных
- Загружен датасет "120 years of Olympic history" с Kaggle
- Проведен первичный осмотр структуры данных (271,116 записей, 15 колонок)
- Выделена только легкая атлетика (38,624 записи, 83 дисциплины)

### 2. Очистка и подготовка данных
- Удалены строки с пропущенными значениями роста и веса (потеряно ~16% данных)
- Рассчитан индекс массы тела (BMI) для каждого спортсмена
- Сохранен чистый датасет для дальнейшего анализа

### 3. Разведочный анализ (EDA)
- **Анализ по полу:** Построены отдельные визуализации для мужчин и женщин
- **Топ-20 дисциплин по росту:** Выявлены дисциплины с самыми высокими спортсменами отдельно для мужчин и женщин
- **Эволюция веса:** Проанализированы изменения веса бегуний на 100 метров за 88 лет (с 1928 по 2016)
- **Корреляционный анализ:** Исследована взаимосвязь роста и веса в 4 ключевых дисциплинах (100 м, марафон, толкание ядра, прыжки в высоту)

## Ключевые выводы:

### Основные результаты анализа:

1. **Четкая кластеризация дисциплин:**
   - **Метатели ядра** — самые тяжелые (♂ ~115 кг, ♀ ~85 кг) и мощные спортсмены
   - **Марафонцы** — самые легкие (♂ ~60 кг, ♀ ~50 кг) и выносливые
   - **Прыгуны в высоту** — самые высокие (♂ ~188 см, ♀ ~178 см)
   - **Спринтеры** — оптимальное сочетание роста и веса для взрывной скорости

2. **Эволюция антропометрии:**
   - За 88 лет (1928-2016) вес бегуний на 100 метров увеличился на **3.1 кг**
   - Это говорит о профессионализации спорта и увеличении требований к силовой подготовке

3. **Ограничения данных:**
   - Выявлена проблема пропусков в данных (особенно для ранних Олимпиад)
   - В анализ включены только спортсмены с полными антропометрическими данными

### Практическая ценность:
-  **Готовый чистый датасет** для дальнейшего машинного обучения
-  **Наглядные визуализации**, демонстрирующие различия между дисциплинами
-  **База для предиктивной модели** (предсказание дисциплины по росту и весу)
-  **Возможность определить для спортсмена дисциплину легкой атлетики, в которой он сможет показать наилучшие результаты, основываясь на его антропометрических данных**
-  **Понимание эволюции спорта** за более чем вековую историю
